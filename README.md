# Projeto de implementação da primeira rede neural adversária

## 📒 Descrição
O Projeto visa a criação de imagem a partir da rede adversária. Com base em uma dataset fashion-mnist. O Projeto tem objetivo principal gerar uma base de estudos para impleentação de outras e usando outras tecnologias.
## 🤖 Tecnologias Utilizada
Foi usado a linguagem python e a biblioteca Torch

## 🧐 Processo de Criação
Foi feito um estudo a partir do video do canal Programação Dinamica
